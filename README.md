## Requirements

* silverstripe/recipe-cms: ^1@dev || ^4@dev
* dnadesign/silverstripe-elemental: ^2@dev || ^3@dev
* dnadesign/silverstripe-elemental-list: ^1.1@dev

## Installation

`composer require dynamic/silverstripe-elemental-tabset`
